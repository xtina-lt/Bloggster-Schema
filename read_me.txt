BLOB stands for Binary Large Objects
It can be used for storing binary data rather than TEXT that stores only large text.
Stores: pictures, videos, sounds and programs.

Can alternately use VARCHAR and file path info.

USERS

- many USERS can view multiple BLOGS
- get information about use

BLOGS

ADMINS

- can have many ADMINS, get their id's from USER

FILES

POSTS
